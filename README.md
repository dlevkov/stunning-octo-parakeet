# React Presentation (Codespaces-ready)

Quick dev steps:

1. Open the `react-presentation` folder in Codespaces (or open `.devcontainer` to auto-launch).
2. Codespaces will run `npm install` automatically (see `devcontainer.json`).
3. Start dev server:

```bash
npm run dev
```

Open forwarded port 5173 to view the app. Click the `English` or `Russian` buttons to load presentations.
